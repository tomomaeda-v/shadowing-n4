# 🎧 Shadowing N4

インドネシア人学習者向けの JLPT N4 レベル・シャドウイング練習アプリです。

## 使い方

- **学習者用アプリ**: `index.html`(UIはインドネシア語)
- **管理画面**: `admin.html`(初期パスワード: `n4admin`)

## 機能

### 学習者向け
- カテゴリ別のN4レベル例文(日本語・かな・ローマ字・インドネシア語訳)
- ネイティブ風TTS音声(MP3)/ 速度調整(🐢ゆっくり)/ 自動リピート(🔁)
- 自分の声を録音してお手本と聴き比べ
- 学習進捗の保存(端末ごと)

### 管理画面
- 文の追加・編集・削除、検索、カテゴリ管理
- `data.json` の書き出し / 読み込み
- パスワード変更

## コンテンツの更新方法

1. `admin.html` で文を編集し「⬇️ data.json を書き出し」
2. GitHub のリポジトリで `data.json` を開き、鉛筆アイコン → 内容を貼り付け → **Commit changes**
3. 数分で全ユーザーに反映されます

※ 新規追加した文の音声は、MP3を作るまでブラウザ読み上げで自動再生されます。
MP3の再生成は `gen_audio.sh`(Open JTalk使用)で行えます。

## 技術構成

- 静的サイト(HTML/CSS/JS のみ、ビルド不要)— GitHub Pages でホスティング
- 音声: Open JTalk(Mei voice)で事前生成した MP3
